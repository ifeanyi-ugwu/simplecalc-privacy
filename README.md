# Privacy Policy for SimpleCalc

**Last updated: February 27, 2026**

## Overview

SimpleCalc is a simple calculator application. Your privacy is important to us.

## Data Collection

SimpleCalc does **not** collect, store, or transmit any personal data. The app:

- Does not require an internet connection
- Does not collect usage analytics
- Does not access any personal information
- Does not use cookies or tracking
- Does not contain advertisements

## Permissions

SimpleCalc does not request any device permissions.

## Third Party Services

SimpleCalc does not use any third-party services that collect data.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.

## Contact

If you have questions about this privacy policy, you can contact us by creating an issue in this repository.
